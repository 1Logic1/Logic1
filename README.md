# Memos Macro (Windows 11)

## Быстрый запуск (из готовых файлов)
1. Скопируйте содержимое папки проекта в любую папку на Windows 11.
2. Запустите `installinterception.exe` **от имени администратора** (установит драйвер Interception).
3. Запустите `Core.exe`.

> Если вы переименуете бинарник в `MemosMacro.exe`, убедитесь, что все сопутствующие файлы (`interception.dll` и др.) лежат рядом с ним.

## Что лежит в корне
- `Core.exe` — основной исполняемый файл.
- `installinterception.exe` — установщик драйвера Interception.
- `interception.dll` — библиотека Interception.
- `main.cpp` — исходный код.

## Примечания
- Если `installinterception.exe` не запускается или не устанавливает драйвер, попробуйте запустить его снова от имени администратора и перезагрузить ПК.
- Проект рассчитан на Windows 11.
